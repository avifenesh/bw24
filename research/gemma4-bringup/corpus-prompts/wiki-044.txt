--- title: Arctic Inference type: entity status: active tags: [inference, serving, vllm, snowflake, speculative-decoding, parallelism, open-source] sources: [] updated: 2026-06-29 --- # Arctic Inference **Arctic Inference** is an open-source vLLM plugin from **Snowflake AI Research** (repo `snowflakedb/ArcticInference`, `pip install arctic-inference[vllm]`) that bundles several of Snowflake's production inference optimizations into a single deployment.[^1][^2] It integrates via vLLM's custom-plugin feature: once installed it auto-patches vLLM so the user keeps the familiar vLLM APIs/CLI, while gaining four headline features that compose together — **Shift Parallelism**, advanced **speculative decoding** (suffix decoding + an LSTM draft model), **SwiftKV** prefill-compute reduction, and optimized embedding inference.[^2][^3] The accompanying paper is *Arctic Inference with Shift Parallelism: Fast and Efficient Open Source Inference System for Enterprise AI* (arXiv 2507.11830, Rajbhandari/Hidayetoglu/Qiao/Wang/Yang/Rasley/Wyatt/He, July 2025), and the system already powers production workloads in Snowflake Cortex AI.[^1][^2] The defining piece is **Shift Parallelism**: rather than statically picking one parallelism strategy, it dynamically switches per forward pass between **tensor parallelism** (TP, best low-TPOT generation on small batches) and **Arctic Sequence Parallelism** (a.k.a.
