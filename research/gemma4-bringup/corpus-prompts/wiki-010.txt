--- title: Hierarchical Quadratic Programming type: concept tags: [robotics, control, optimization, inverse-kinematics, whole-body-control] sources: [] updated: 2026-06-30 --- # Hierarchical Quadratic Programming Hierarchical quadratic programming (HQP) is an optimization framework for solving a set of conflicting linear objectives and constraints arranged in a **strict priority order**. Instead of folding every task into one weighted cost (where high-weight tasks merely dominate but never strictly override), HQP enforces a true lexicographic hierarchy: a lower-priority task may only use the residual freedom left over after every higher-priority task has been satisfied as well as it can be. This is implemented as a **cascade of quadratic programs (QPs)** — each level minimizes the slack on its own tasks subject to the constraint that the optimal slacks achieved at all higher levels are preserved.
