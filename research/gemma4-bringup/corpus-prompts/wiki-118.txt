--- title: Elastic Fabric Adapter (EFA) type: entity tags: [aws, networking, hpc, distributed-training, rdma, libfabric] sources: [] updated: 2026-06-29 --- **Elastic Fabric Adapter (EFA)** is an AWS network device that attaches to Amazon EC2 instances to accelerate distributed AI/ML training and HPC workloads that need tight inter-node communication.[^1] Functionally it is AWS's equivalent of InfiniBand: alongside the standard ENA (Elastic Network Adapter) TCP/IP path it exposes an **OS-bypass** path through the `libfabric` API, letting applications talk to the network hardware directly without going through the kernel, which cuts latency and CPU overhead for collective operations.[^1][^2] On GPU instances EFA pairs with **GPUDirect RDMA** so the NIC reads and writes GPU memory directly, bypassing the CPU for inter-node transfers.[^3] It is the transport that NCCL (and frameworks built on it) ride on when scaling training across many nodes on AWS.
