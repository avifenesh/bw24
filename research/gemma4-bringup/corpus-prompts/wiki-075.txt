--- title: agent knowledge base (in-repo agent corpus) type: concept tags: [agents, knowledge-base, agents-md, claude-md, llms-txt, learn, in-repo-docs] sources: [] updated: 2026-06-29 --- # agent knowledge base (in-repo agent corpus) An **agent knowledge base** is a set of files committed alongside a project that exists for AI coding agents to read, rather than for the human-facing build or runtime. It captures the project context, conventions, and background research an agent needs so it does not have to re-derive that knowledge from scratch on every task. The pattern overlaps several emerging conventions: `AGENTS.md` (an open, markdown format that sits "at the top of the conversation history, right below the system prompt" and carries the extra detail — build steps, tests, conventions — that would clutter a README or isn't relevant to human contributors)[^1][^2], the tool-specific `CLAUDE.md` equivalent, and `llms.txt`, a website-level file meant to help language models understand and prioritise a site's content the way `robots.txt` and `sitemap.xml` serve crawlers.[^3] An arXiv study from early 2026 ("Evaluating AGENTS.md: Are Repository-Level Context Files Helpful for Coding Agents?", 2602.11988) treats these repository-level context
