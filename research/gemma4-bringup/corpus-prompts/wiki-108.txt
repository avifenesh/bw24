--- title: Itch (Outward Idea Finder) type: entity status: active tags: [itch, idea-finder, python, agent-cli, web-research] sources: [itch-idea-finder, collide-itch-feature, atrium-life-dashboard] updated: 2026-06-29 --- # Itch (Outward Idea Finder) Itch (`~/projects/itch`) is an outward idea scout. It compiles a de-dup profile of what the owner already does — local `~/projects` dir names, `gh repo list`, starred repos as a taste signal, decayed prior-run titles, a decisions ledger, and rules.md — and hands one big prompt to a headless agent CLI that web-searches and returns 6-10 ranked NEW ideas. The portfolio profile is exclusion signal only (the prompt forbids finish/ship/revive proposals); valkey ecosystems are off-limits. Per [[git-state-not-ship-status]], it reads dir names only, never inferring ship status from git. The engine `itch.py` is 2950 lines, stdlib-only, with the backend routed by model-id string (claude CLI / `codex exec --json` / `pi -p`). `itch_ui.py` is 2193 lines, a stdlib ThreadingHTTPServer on port 8799 (always pin `ITCH_PORT`). Data is flat files in `~/.config/itch` (no DB): `decisions.json` is a ledger keyed by lowercase title, with `as_of` = run-stem decay clock; rating 5 = pursuing, 4 =
